# Cardano P2P DApp

Astro + Qwik + Lucid-Cardano lightweight DApp for peer-to-peer commerce on Cardano using a self-governed token.

## 🚀 Quick Start (Codespaces Recommended)

**Best way to develop:** Use GitHub Codespaces (one-click reproducible environment).

1. Open repo → **Code** → **Codespaces** → **Create codespace on this branch**
2. Devcontainer auto-configures Node 20, installs all deps, and loads only essential extensions.
3. Run:
   ```sh
   npm run dev
